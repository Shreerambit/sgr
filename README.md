# README.md

## Project Overview
This is a personal portfolio website for Shreeram Bhajantri, showcasing his education, skills, projects, competitive exam preparations, YouTube channels, and contact information. The site is built with modern web technologies, focusing on responsiveness, performance, and accessibility.

## Tech Stack Used
- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- No frameworks or libraries (pure vanilla)
- Google Fonts (Poppins)
- Formspree for contact form (replace action URL with your own)

## How to Run Locally
1. Clone the repository: git clone https://github.com/yourusername/portfolio.git
2. Navigate to the project directory: cd portfolio
3. Open index.html in your browser.

## How to Deploy on GitHub Pages
1. Create a GitHub repository.
2. Push the code to the repository.
3. Go to repository settings > Pages > Select branch (main) and root directory.
4. The site will be live at https://yourusername.github.io/repository-name.

## How to Update Content Easily
- Personal Info: Edit the relevant sections in index.html (e.g., About, Skills).
- Projects/YouTube Links: Update the placeholders in the respective sections.
- Resume: Replace resume.pdf with your actual file.
- Contact Form: Replace the Formspree action URL in the form tag with your own (sign up at formspree.io).
- Images/Icons: Add files to assets/images/ or assets/icons/ and reference them in HTML.
- Favicon: Replace assets/icons/favicon.ico.
